# 선형대수학 

선형(Linearity): 직선처럼 똑바른 도형 또는 그와 비슷한 성질을 갖는 대상

수학적 정의 : f(x+y) = f(x) + (f(y)  , f(ax) = af(x)

선형 함수의 조건 : 


1) 차수가 2이상 나면안됨 

증명 : f(x) = x^2
f(ax) = f(ax)^2 = a^2x^2
af(x) = ax^2

af(x)랑 f(ax) 랑 안같음. 즉 안됨.


2) 입력이 0 이면 출력도 0

증명 : f( 0 + y) = f(0) + f(y)
f(y) = f(0) + f(y)  
0 = f(0)
